#### 特色及功能

- 上游 stable 分支中设备，只跟进 stable
- 默认密码: 无
- 依赖内核模块时，再考虑编译进固件
- 二进制 golang 应用，尽量不编译进小闪存设备
- K2P blob 的 wifi 采用闭源驱动
